  • HTTPPart

    The parts of a complete HTTP message, either request or response.

    A HTTP message is made up of a request or status line with several headers, encoded by .head, zero or more body parts, and optionally some trailers. To indicate that a complete HTTP message has been sent or received, we use .end, which may also contain any trailers that make up the message.
